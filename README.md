директория ```server/linux-x64``` имеет файл ```ruvds-tcp``` файл.

чтобы получить разрешение на работу с программой:

```chmod 755 ./ruvds-tcp```

чтобы запустить сервер нужно запустить файл:

```./ruvds-tcp```

----------------------------------------

программа попросит ввести ip, а потом порт.
